# Promo Studio — make event promo videos in your browser

A single-file video maker for event promos and save-the-date posts.
No installation, no account, nothing to set up: unzip, then open
**promo-studio.html** in **Google Chrome or Microsoft Edge** (double-click
it). Everything runs on your own computer — nothing is uploaded anywhere.

Made by the NorCal SBDC — the same design system behind our event videos.

## 60-second quick start

1. Open **promo-studio.html** in Chrome or Edge.
2. **Brand panel (left):** pick a heading + body font, set your three
   brand colors, and load your logo (paste a URL to a PNG on your
   website, or upload the file — upload is the most reliable). A white
   version for dark slides and a color version for light slides look best.
3. Paste your **event link** (use a short link — it shows on the video)
   and click **Apply to link slots**.
4. **Slides (middle-left):** the starter storyboard is
   hook → save-the-date → end card. Click a slide to edit its text on the
   right; add slides (agenda, quote, big number, photo…) as needed.
5. Press **▶** to watch it loop. Fix anything that wraps badly (there's a
   "Text size" slider for long titles).
6. **Export MP4** (top right) downloads the finished video —
   1080×1080 by default, ready for Instagram, LinkedIn, or Facebook.
   **Download PNG frame** saves the current frame as a static graphic —
   an instant matching image for posts, newsletters, or stories.

## A storyboard that works (about 15–25 seconds)

1. **Statement** — one-line hook (what attendees get, not the event name)
2. **Save the date** — the date tile, title, time, link footer
3. **Agenda list** — 3–4 lines, "topic | time"
4. **End card** — logo, "Register today", your short link

Rules of thumb from our design system: sentence-case titles, ALL-CAPS only
in the small kicker labels, no exclamation marks, no emoji, one accent
color. Short links only on cards — never a long registration URL.

## Tips

- **Fonts** load from Google Fonts (needs internet the first time). A
  serif heading (Lora, Playfair Display) + clean sans body (Inter,
  Source Sans 3) is a safe, classy pair.
- **Logo won't load from URL?** Some websites block image reuse.
  Download the logo file, then use "upload a file" — that always works.
- **Photos:** add a "Photo / background image" slide and upload an image.
  Use the overlay to keep text readable. Use photos you have rights to.
- **Music:** optional MP3/WAV bed with automatic fade in/out. Use
  licensed or royalty-free tracks.
- **Aspect:** square 1:1 for feeds, 9:16 for Reels/Stories, 16:9 for
  YouTube/screens — switch it in the top-left dropdown any time.
- **PNG button = free graphics.** Every slide doubles as a static social
  graphic — scrub to it and click Download PNG frame.

## Requirements & limits

- Chrome or Edge on a computer (the MP4 encoder is built into those
  browsers). Safari/Firefox can preview and save PNGs, but not export MP4.
- Internet is only needed for fonts and logo-by-URL; everything else is
  offline.
- Videos export at 1080p, 30fps MP4. Standard Chrome/Edge on Windows and
  Mac encode H.264, which plays everywhere. If a platform ever rejects
  an upload, re-export from an up-to-date Chrome on Windows/Mac.
