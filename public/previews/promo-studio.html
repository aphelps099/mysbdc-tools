Promo Studio

Make on-brand event promo videos and social graphics in your browser. One file, no installation, no accounts — your event data never leaves your computer.

Download Promo Studio (ZIP) Design-system guide (MD)
Quick start
  1. Unzip, then open promo-studio.html in Chrome or Edge.
  2. Set your fonts, three brand colors, logo, and event link in the Brand panel.
  3. Edit the starter storyboard: hook → save-the-date → end card.
  4. Export MP4 for the video, or Download PNG frame for a matching static graphic.
Make it yours with Claude
  1. Load EVENT-PROMO-SYSTEM.md into Claude Design (or Claude/Cowork).
  2. Run an "exploration": Claude proposes brand token sets and mocks the cards in your identity.
  3. Ask for the handoff — a small JSON of your final tokens.
  4. Type those values into Promo Studio's Brand panel. Done — your network's design system, working.

Requires Chrome or Edge on a computer for MP4 export · Internet needed only for Google Fonts and logo-by-URL.
