Implementation Notes

For Gmail / Google Workspace:

Settings → See all settings → General → Signature. Paste HTML or use rich text editor.


For Outlook:

File → Options → Mail → Signatures. Create new, paste content.


For Apple Mail:

Mail → Preferences → Signatures. Create in TextEdit as HTML, paste.


Image hosting:

Host logo and gradient images on norcalsbdc.org for reliable loading. Recommended: logo at 160px wide (2x for retina).


Fallback:

S1, S5, and S8 work without images. Use these if email clients strip formatting.
